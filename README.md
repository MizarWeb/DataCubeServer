# DataCubeServer

## Installation 
```
git clone https://github.com/MizarWeb/DataCubeServer.git
cd DataCubeServer
vim cubeExplorer.properties and set the same <path> (eg : /tmp/tests/) for workspace and workspace_cube
mkdir <path>/private
mkdir <path>/public>
mvn install
java -jar target/cubeExplorer-1.0.0-SNAPSHOT.jar
```
